Free static hosting directly from your GitHub repository. Perfect for client-side applications.

Netlify

Drag and drop deployment with continuous deployment from Git repositories.

  2. Drag your project folder to the deploy area
  3. Get an instant live URL

Vercel

Zero-configuration deployment with excellent performance and global CDN.

  4. Deploy automatically on every push
